Configuration
==============

Here we explain the different configuration files and all adjustable parameters.

From here on we will refer to the work directory in which the work is carried out as the palantiri main workdir.
To create the palantiri main workdir type the following command in your console::

  palantiri_init foldername

Palantiri uses three different configuration files, which specify the user input.
